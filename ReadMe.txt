Библиотека, для создания xlsx-документа по шаблону.

Для работы необходимо:
1. Создать xlsx-шаблон, состоящий из:
	a) Линейных переменных {переменная}
	б) Переменных массива
		1) Горизонтальный массив surname_g
		2) Вертикальный массив   surname_v

2. Организовать входной массив, соответствующий, уже заданному шаблону
3. Создать объект класса с параметрами
	а) Путь к шаблону
	б) Путь к новому файлу
	в) Входной массив
4. Вызвать метод "getNewXLSX()"

Возможности библиотеки:
1. Заполнение ячеек как строковыми значениями так и числовыми;
2. Линейные переменные могут быть в составе другого сторокого значения;
3. В ячейке могут находиться несколько линейных переменных; 
4. Заполнение ячеек массивом данных по горизотале,со смещением ячеек вправо;
5. Заполнение ячеек массивом данных по вертикале, со смещение срок вниз;
6. Если в строке несколько ячеек со значением, которое соответствует
	вертикальному массиву - строки смещаются вниз равномерно;
7. Значения ячеек, соответствующие вертикальным и горизонтальным массивам 
	состоят строго из переменных |___surname_v___|;
